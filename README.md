# Avito Radar Bot — парсер объявлений Avito

Telegram-бот для поиска и анализа объявлений на Avito.

## Фичи
- Поиск объявлений по любому запросу
- Excel-отчёт: Название, Цена, Локация, Ссылка
- Оплата Telegram Stars (300⭐/мес)
- Бесплатный триал 3 дня
- Rate limit 8 сек (безопасный для Avito)

## Стек
- Python + aiogram 3.x + httpx + BeautifulSoup4
- openpyxl — Excel
- Telegram Stars API — оплата

## Деплой
```bash
BOT_TOKEN="ваш_токен" python bot.py
```
